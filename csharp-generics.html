Generic classes and methods combine reusability, type safety and efficiency
Generics are most frequently used with collections and the methods that operate on them.
Version 2.0 of the .NET Framework class library provides a new namespace, System.Collections.Generic

By creating a generic class, you can create a collection that is type-safe at compile-time.

Any reference or value type that is added to an ArrayList is implicitly upcast to Object.

Both the casting and the boxing and unboxing operations decrease performance.

The other limitation is lack of compile-time type checking

by Generic List, you can create a list that is not only safer than ArrayList, but also significantly faster.
